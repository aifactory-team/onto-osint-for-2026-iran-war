# Phase 3 온톨로지 분석 — 2026-06-11 (Day 104)

## 분석 요약

6월 11일은 전쟁의 결정적 전환점이다. 6월 9일 성립된 상호 공격 중단이 48시간도 채 되지 않아 완전히 붕괴하였고, 양측 모두 전쟁 개시 이후 가장 공격적인 군사 행동을 감행하였다.

## 1. 상호 중단 붕괴의 의미

### 1.1 타임라인
- **6/9**: 이란-이스라엘 상호 공격 중단 성립 (ent-544)
- **6/10**: 상호 중단 2일째, "취약" 상태 (트럼프 'much more aggressive' 톤 변화)
- **6/11**: 완전 붕괴 — 미국 이란 본토 직접 폭격 (ent-557)

### 1.2 구조적 실패 원인
상호 중단은 세 가지 구조적 결함이 있었다:

1. **비공식성**: 서면 합의 없이 양측 자발적 중단에 의존
2. **범위 제한**: 이란-이스라엘 직접 교전만 대상, 미국-이란/레바논 전선 미포함
3. **이행 메커니즘 부재**: 위반 판정/제재 장치 없음

Apache 헬리콥터 격추(추정)가 에스컬레이션 트리거로 작용하여, 미국이 49발 토마호크 + 전투기 폭격으로 대응하면서 중단은 완전히 무너졌다.

## 2. 에스컬레이션 사이클 분석

### 2.1 인과 체인
```
Apache 격추 (trigger)
  -> 미국 49 토마호크 + 전투기 (ent-557)
    -> 이란 4개국 보복 공격 21건 (ent-558)
      -> 미 대사관 철수 명령 (ent-559)

```

### 2.2 에스컬레이션의 질적 변화
이전의 미-이란 교전과 비교하여 세 가지 질적 변화가 관찰된다:

1. **규모**: 49발 토마호크는 단일 작전 기준 전쟁 이후 최대 규모. 테헤란에서 40마일 거리 표적은 이란 수도 근접 공격의 의도적 시그널.
2. **동시성**: 이란의 21건 보복이 4개국(이라크/쿠웨이트/바레인/요르단)에서 동시 발생. 이는 사전 계획된 보복 계획의 실행을 시사.
3. **상황실 결정**: 트럼프가 밴스/쿠슈너/위트코프와 상황실에서 직접 공습을 승인. 6/8 'I call all the shots' -> 6/11 실제 대규모 공습 승인으로 격상.

## 3. 대사관 철수 — 사전 배치 시그널

### 3.1 조치 내용
- 바그다드: **명령 철수** (ordered departure)
- 바레인/쿠웨이트: **자발적 출국 허용** (authorized departure)
- 미군 부양가족: **자발적 퇴거**

### 3.2 역사적 선례와 의미
대사관 철수 명령은 대규모 군사 작전의 사전 지표로 기능한다:
- 2003년 이라크 침공 전 바그다드 대사관 철수
- 2011년 시리아 내전 시 다마스쿠스 대사관 폐쇄

바그다드 **명령 철수**와 바레인/쿠웨이트 **허용 철수**의 구분은, 이란 보복 타격의 직접 위험 지역(이라크)과 잠재적 위험 지역(걸프)을 미국이 구분하고 있음을 보여준다. 이는 더 큰 규모의 공습이 임박했음을 시사하는 사전 배치 시그널이다.

- **6/11: 49 토마호크 -> MoU 프레임워크 사실상 사망**

4. 트럼프 '폭격' 최후통첩은 협상이 아닌 강압의 프레임

### 5.3 헤그세스 독트린

## 6. 이란 내부 역학

### 6.1 IRGC 에스컬레이션 vs 외교 트랙
6/11의 이란 보복은 두 가지 해석이 가능하다:

1. **사전 계획설**: IRGC가 상호 중단과 무관하게 보복 계획을 유지했으며, 미국 공습이 실행 트리거 제공
2. **자동 보복설**: 이란 군사 독트린에 따른 자동적 비례 보복 (21건은 49발 토마호크에 대한 계산된 비례 대응)

### 6.2 호르무즈 "완전 봉쇄" 주장
IRGC의 완전 봉쇄 주장 vs CENTCOM의 부인은, 양측의 호르무즈 통제 내러티브 싸움의 연장이다. 실질적 봉쇄 수준은 부분적이나, IRGC는 에스컬레이션 메시지로서 "완전" 봉쇄를 주장한다.
